# github-issues-viewer

Github `issues` 탭에서 다음과 같은 불편한 점을 느껴, 편리하게 사용하고자 제작한 이슈 뷰어입니다.

- 이슈 간 빠른 전환이 어려움
- 사용성을 저해하는, 너무 많은 정렬 & 필터링 요소
